Задание №1

Напишите приложение, где работали бы два компонента:
Упражнение select. На вход компонент принимает массив ответов (строка + правильность ответа). Пример: [{value: "apple", correct: false}, {value: "pear", correct: true}, {value: "pineapple", correct: false}]. При этом, компонент на страницу может добавляться как через ng-repeat и передачу данных из контроллера, так и вручную: <vt-select answers="{value: "apple", correct: false}"></vt-select>. Упражнения могут располагаться в разных частях страницы и находиться внутри изолированных scope.
Виджет для отображения количества очков. Отображает всего 2 числа: общее количество упражнений select на странице; количество упражнений, где выбран правильный ответ.

Задание №2

Напишите компонент, который реализует виджет для изучения слов. На вход компонент принимает массив слов и переводов. Пример: [{word: "apple", translation: "яблоко"}, {word: "onomatopoeia", translation: "звукоподражание"}]. Компонент должен выводить на экран в один момент времени только одну пару слов "apple — яблоко", переключение к следующей паре происходит при нажатии на пробел.
